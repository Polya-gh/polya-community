# Submissions

Problem sheets submitted to the Polya platform, organized to match the platform's URL structure.

## Structure

```
/submissions
  /mathematics
    /calculus
    /algebra
    /geometry
  /physics
    /mechanics
    /optics
    /thermodynamics
```

Each file maps to a subtopic at `polya.live/{subject}/{topic}/{subtopic}`.

## File Naming

```
{subtopic}_{n}.{extension}
```

**Examples:**
```
linear-motion_1.tex
linear-motion_2.typ
calculus-1_1.tex
```

## Submitting

1. Fork this repository
2. Place your file in the correct subject/topic folder
3. Follow the template and variable spec in [/templates](../templates/README.md)
4. Open a pull request

Merged submissions are automatically imported into the platform.
