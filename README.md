# Polya Community

This is the community repo for [Polya](https://polya.live).

This is where the problem library is built. Anyone can contribute problems, suggest improvements, or report errors.

---

## Repository Structure

```
/community
  /templates        ← LaTeX and Typst templates for writing problems
  /submissions      ← submitted problems
```

---

## How to Contribute

There are two ways to submit a problem:

**Via the web** — visit [polya.live/contribute](https://polya.live/community/contribute), fill out the form, and submit. A pull request is opened automatically, no git knowledge required. Available from Scholar and above.   

**Via template** — download a template from `/templates`, fill it out locally in your editor, and open a pull request against the [submissions repository](https://github.com/polya/submissions).

All submissions are reviewed manually before being imported.

---

## Guidelines

- Problems must be original or sourced from public domain / freely reproducible competition archives
- Always cite the source if the problem is from a known competition or textbook
- The answer must be exact and verifiable
- A worked out solution is required; hints are optional but encouraged

---

## Reporting Errors

If you find an incorrect answer, a broken solution, or a poorly stated problem on the platform, open an issue in this repository with the file name and a description of the error.
